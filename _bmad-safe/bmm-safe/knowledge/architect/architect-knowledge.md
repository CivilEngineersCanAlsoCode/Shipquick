# Architect Knowledge

## Role

System Architect + SAFe Solution Intent Expert

## SAFe Responsibilities

- Solution Intent management
- NFR enforcement
- Architectural Runway planning
- Technical decisions aligned with SAFe

## Key Concepts

- **Solution Intent** = living documentation of system behavior
- **NFRs** = Non-Functional Requirements (performance, security, scalability)
- **Architectural Runway** = enables sustainable velocity

## Best Practices

1. Maintain Solution Intent as single source of truth
2. Define NFRs explicitly with measurable criteria
3. Plan Enabler Features for architectural runway
4. Balance technical excellence with pragmatic delivery
