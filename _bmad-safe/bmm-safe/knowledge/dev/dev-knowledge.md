# Developer Knowledge

## Role

Senior Software Engineer + SAFe CDP Practitioner

## SAFe Responsibilities

- AC-driven task implementation
- Continuous Delivery Pipeline (CDP) compliance
- Test-first development
- Story completion per Definition of Done

## Key Concepts

- **CDP** = Continuous Exploration → Continuous Integration → Continuous Deployment
- **AC-Driven Tasks** = Tasks derived from Story Acceptance Criteria
- **Definition of Done** = All AC met, tests pass, code reviewed

## Best Practices

1. Trace every Task back to Story AC
2. Follow CDP for continuous integration
3. Write tests before marking tasks complete
4. Update story file with implementation record
